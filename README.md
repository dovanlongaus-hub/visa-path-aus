# Úc Di Trú AI – visa-path-aus

Nền tảng tư vấn di trú Úc thông minh dành cho người Việt Nam.

## Tech Stack
- React 18 + Vite
- Tailwind CSS + shadcn/ui
- Base44 Platform (BaaS)
- Lucide React Icons
- React Router DOM

## Tính năng chính
- 🤖 Tư vấn AI 24/7 về visa, di trú, điểm EOI
- 🗺️ Lộ trình PR cá nhân hoá (500→85→189/190/491)
- 📝 Checklist & MyPlan AI
- 📄 Biểu mẫu di trú tự điền
- ✨ EOI & CV Generator AI
- 📊 Knowledge Base & FAQ
- 🔔 Smart Alerts cảnh báo visa

## Tạo bởi
DVLong & Genetic AI
